# ⚡ REALGODS - E-COMMERCE LAB

**Realgods** (formerly known as Loyal) is a high-end streetwear digital experience. This platform is designed to showcase "oversized" collections with a minimalist, dark, and industrial aesthetic, optimized for quick conversions and a seamless user experience.

## 💀 The Concept

Realgods isn't just a store; it's a lab. The UI reflects the "Street Food Lab" and "Oversize Culture" vibe, using a deep-dark palette with neon orange accents and glow effects to highlight the exclusivity of each piece.

## 🚀 Key Features

- **Strategic Navigation**: A compact and efficient category selector for seamless browsing between drops.
- **Product Detail Engine**: High-priority modals providing deep dives into fabric specs, sizing, and "oversize" fits.
- **Advanced Cart Logic**:
    - **Smart Grouping**: Automatically groups identical items while maintaining unique customization notes.
    - **Item Quantities**: Full control over product volume within the cart.
    - **One-Tap Clear**: Instant reset for a fresh shopping session.
- **Custom Order Notes**: Allows customers to specify custom requirements directly on the product detail modal.
- **Multi-Channel Checkout**:
    - **WhatsApp Direct**: Order formatting for immediate communication with the sales team.
    - **In-Store Ticket**: Digital receipt generation for local pickups or physical showroom visits.
- **Optimized UI**: Complex `z-index` management to ensure a zero-overlap experience between floating elements and high-priority modals.

## 🛠️ Technical Stack

- **React.js**: Core engine for the SPA (Single Page Application).
- **JavaScript (ES6+)**: Custom logic for cart state management and WhatsApp API integration.
- **CSS3 / Styled-Thinking**: Custom glow effects, backdrop blurs, and responsive grid layouts.
